# FIZZBUZZ-EXCEPCIONES
revisar si un numero es multiplo de 3 imprime fizz
revisar si un numero es multiplo de 5 imprime buzz
revisar si un numero es multiplo de 3 y 5 imprime fizzbuzz
en otro caso imprime el numero
consideracion:
para saber si un numero es multiplo de otro se usa el operador matematico modulo(%) para ver que devuelve
si el residuo = 0 es multiplo y != 0 no es multiplo
